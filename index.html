What is Monerosci?

Monerosci is an analytical and data exploring tool developed for the Monero Blockchain. This is still developing tool and more features will add in future. For the moment this include Monerosci-parser which is a module that convert all the blockchain data to Monerosci data format. And also this include python-interface to import monerosci as a library for Python. Overall development is based on C++ programming language.
